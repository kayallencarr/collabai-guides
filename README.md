# collabAI Guides

A collection of companion guides for [collabAI](https://joincollabai.com) members — walkthroughs on building real things with AI partnership.

These guides go beyond prompts and templates. They show you how to use AI to actually ship projects: deploy websites, build automations, set up cloud infrastructure, and more. Written for entrepreneurs, not developers.

## Guides

- [How to Deploy Your Website with Claude Code](guides/deploy-your-website.md) — Get your AI-built landing page live on the internet with help from Claude Code. Vultr hosting, DNS, SSL — all handled by your AI developer.
- [Getting Started with Claude Agents](guides/getting-started-with-claude-agents.md) — **Start here.** A 30-minute quickstart for setting up your first scheduled Claude agent. Works for any use case.
- [Automated Claude Agents (Detailed)](guides/automated-claude-agents.md) — The full walkthrough. Set up agents that triage your email, check your Slack, and run on reliable schedules — all in the cloud, no computer required.
- [Autonomous AI Sessions](guides/autonomous-ai-sessions.md) — How to give your AI partner its own time to think and create. Wake up to business recommendations and reflections written while you slept.
